# Align Property

Learn how to use align property for flexbox and grid layout

## Type Rules

There's some type you can use for align the items:

`ai-` : This type is used to change `align-items` property

`ac-` : This type is used to change `align-content` property

`as-` : This type is used to change `align-self` property
