# Color

In this page you will learn how to change color for text, background, border ( maintained ), and you can also learn for using `css variable` and apply it into element.

## Text Color

You can use `tc-` type to change the color of the text of the element.

### Usage Example

This simple code you can copy and try it :

<Tabs>
<TabItem value='html' label='index.html'>

Add some element in html.

```html
<!-- Adding color css class -->
<header>
  <h1 class="tc-white">Some Title</h1>
  <p class="tc-ddd">Some text</p>
</header>
<!-- Color from CSS -->
<header>
  <h1 class="tc-[primary]">Some Title</h1>
  <p class="tc-[secondary]">Some text</p>
</header>
```
</TabItem>

<TabItem value='css' label='style.css'>
Add some css variable that can be called in html.

```css
:root {
  --primary: #09cf5c;
  --secondary: #c45210;
}
```
</TabItem>
</Tabs>


## Background Color

We define `background-color` not as `bc` type, but its merged it with `bg` type. So, if you want to change the background color, you can just use `bg` type.

### Usage Example

This simple code you can copy and try it :

<Tabs>
<TabItem value='html' label='index.html'>
Add some element in html.

```html
<section class="bg-black">
  <div class="box-[box-size] bg-red m-0.8em"></div>
  <div class="box-[box-size] bg-[primary] m-0.8em"></div>
  <div class="box-[box-size] bg-[gradient-back] m-0.8em"></div>
</section>
```

:::info[What is box?]

`box-` type is TenoxUI's type that can change `width` and `height` at the same time.

:::


</TabItem>

<TabItem value='css' label='style.css'>
Add some css variable that can be called in html.

```css
:root {
  --primary: #09cf5c;
  --box-size: 50px;
  --gradient-back: linear-gradient(to right, red, blue);
}
```

:::tip[using background-image]

You can also define image for the background or `background-image` and call it using `bg-` type:

```html title="index.html"
<div class="bg-[back] m-0.8em"></div>
```
```css title="style.css"
:root {
  --back: url("image.png");
}
```

:::

</TabItem>
</Tabs>

## Supported Inline Color

There's a few color that you can use directly on the className :

### 1. Color Name

All color that available in css is valid on TenoxUI. Like, red, blue, yellow, lightblue, tomato, and more, its all valid in TenoxUI and you can just type it this way:

```html title="index.html"
<div class="bg-red">
  <h1 class="tc-blue">Some Text</h1>
</div>
```

Color name will always be string.

```bash
pkg install tenoxui
```

### 2. HEX Color

HEX color is also valid in TenoxUI

```html title="index.html"
<div class="bg-red">
  <h1 class="tc-blue">Some Text</h1>
</div>
```
