# Flexbox
## Intro 

Flexbox is a one-dimensional layout system that we can use to create a row or a column axis layout. It makes our life easier to design and build responsive web pages without having to use tricky hacks and a lot of float and position properties in our CSS code.

## Display Flex

To setting display property you can use `display-{value}` to the display property. In this case you can use `display-flex` : Set `display: flex;` for element.

## Flex Direction



<Tabs>
  <TabItem value="width" label="Width">

### Width Property

`w-` : Set `width` to the element.

`w-mn-` : Set `min-width` to the element.

`w-mx-` : Set `max-width` to the element.

</TabItem>

<TabItem value="height" label="Height">

### Height Property

`h-` : Set `height` to the element.

`h-mn-` : Set `min-height` to the element.

`h-mx-` : Set `max-height` to the element.

</TabItem>

</Tabs>


:::tip

You can also use `% (percent)` as the unit. Such as `w-100%` and so on.`

:::
