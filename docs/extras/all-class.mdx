## Custom Type and Property ✨

You can make your own type and property by using [addType and defineProps](/docs/function/addtype-and-defineprops) function. Here's simple usage :

```html title="html"
<div class="my-box-type-300px"></div>
<div class="my-bg-type-blue"></div>
```
```js title="javascript"
defineProps({
  boxs: ["-webkit-box-sizing"],
  "my-box-type": ["width", "height"],
  "my-bg-type": ["background-color"]
})
```
